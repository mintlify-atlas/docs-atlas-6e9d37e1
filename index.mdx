WASI Testsuite provides a comprehensive set of tests for the [WebAssembly System Interface (WASI)](https://github.com/WebAssembly/WASI) and a Python-based test runner that validates WASI compliance across any WebAssembly runtime. It covers WASI preview 1 and the forthcoming WASI preview 3.

## Get up and running

<Steps>
  <Step title="Clone the repository">
    Use the `prod/testsuite-base` branch which includes precompiled test binaries — no compiler toolchain required.

    ```bash
    git clone --branch prod/testsuite-base https://github.com/WebAssembly/wasi-testsuite
    ```
  </Step>
  <Step title="Install the test runner">
    Install Python 3 and the test runner dependencies.

    ```bash
    python3 -m pip install -r test-runner/requirements.txt
    ```
  </Step>
  <Step title="Run the test suite">
    Execute all tests against all available runtimes, or target a specific one.

    ```bash
    # Run against all available runtimes
    ./run-tests

    # Run against a specific runtime
    ./run-tests --runtime-adapter adapters/wasmtime.py
    ```
  </Step>
</Steps>
